Filter Data Flow

1 · User Input
FilterBar.vue
User changes Period / Warehouse / Category / Status dropdown via v-model
›
2 · State
useFilters.js
Shared singleton composable updates reactive refs. All views watch these refs.
›
3 · API Call
api.js (Axios)
getCurrentFilters() builds query params: ?warehouse=X&category=Y&month=Z
›
4 · Backend
FastAPI Route
apply_filters() loops in-memory list, skips params set to 'all'
›
5 · Validation
Pydantic Model
Filtered list serialized via response_model, types enforced
›
6 · Render
Vue View
Reactive ref updated → computed properties recalculate → template re-renders
